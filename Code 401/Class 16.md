> ## 1. Spring Security overview

**Spring Security Solves two main Problems**
  
  1. authentication: check if the user is allow for access
  2. authorization: check if the authenticated user allows editing 

**The library used in Spring authentication & authorization called:** Spring Security where it can be found in Spring Initializer website or adding it to your dependencies

    `<dependencies>
        <dependency>
            <groupId>org.springframework.boot</groupId>
            <artifactId>spring-boot-starter-web</artifactId>
        </dependency>
    </dependencies>`
    
**Types of Security to consider**

1. basic : security for your app
2. csrf : Cross-site request forgery for forigen API's
